# Language Learning Website

## Flashcard for vocabulary memorization

## Rearrange Sentences for grammar practice

## Immersion Conversation mode for thought training

## Switch between Audio and Text mode

## Pronounciation Training

# Website Flow:
- Homepage: Languages List
    - Currently supports Chinese HSK 1 to 6, French
- On clicking into a language
    - Tiles: flashcards, sentence rearrangement game, conversation immersion, pronounciation training
    - Flashcard Tile
        - Shows chinese word and pinyin, clicking on it flips it to show the english word. On viewing 20 words it unlocks game mode, where you are shown 4 options and pick the correct translation.
    - Sentence Rearrangement Game
        - Sentences are scrambled, rearranging them brings up the translation and then the next scrambled sentence.
    - Conversation immersion
        - Prompts are provided, user talks for a minute and is then provided with a translation of what they said.
    - Pronounciation training
        - Record voice and compare it to the prerecorded correct sound.
